# Insomnia Helper App

An Android app to help people who struggle with sleep and stress.

## Features
- Severity quiz that recommends solutions based on your level
- Relaxing ocean and rain sounds to help you sleep
- Breathing exercise animation that expands and contracts
- Sleep journal to write thoughts and track mood
- Daily sleep hour tracker to monitor improvement

## Built With
- Java
- XML
- Android Studio

## How to Run
Open in Android Studio and run on an Android device or emulator
